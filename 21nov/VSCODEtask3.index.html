VISUAL STUDIO CODE

Definition :

VS Code (Visual Studio Code) is a free, lightweight, and open-source code editor developed by Microsoft. It is widely used for software development due to its powerful features and extensive customization options.

Key Features :

    1. Multi-language support :Supports many programming languages out of the box (e.g., JavaScript, Python, C++, etc.) with the ability to add more via extensions.

    2. Built-in Git integration :Provides seamless version control with Git. Extensions Marketplace: Offers a wide range of extensions to enhance functionality, such as themes, debuggers, and language-specific tools.

    3. IntelliSense :Provides smart code completion, syntax highlighting, and error detection.

    4.IntelliSense :Provides smart code completion, syntax highlighting, and error detection.

    5.Integrated terminal:Allows you to execute commands without switching to an external terminal.

    6.Cross-platform:Works on Windows, macOS, and Linux.

VS Code has gained popularity for its speed, flexibility, and user-friendly design, making it a go-to tool for both beginners and professional developers.

Why we use VS Code ?

    Lightweight yet Powerful : Unlike full IDEs, it opens quickly and uses fewer resources.
    Versatile for multiple workflows:Works for everything from web development to embedded systems.
    Active community:Constant updates and contributions.

How to get Started :

    1. Download & Install:
  • Download VS Code from the officcial site

    2. Add Extensions:
  • Open the Extensions Marketplace (Ctrl+Shift+X or Cmd+Shift+X).
  • Search and install extensions for your desired tools/languages.

    3. Customize Settings :
  • Access settings via Ctrl+, or Cmd+, to adjust themes, keyboard bindings, and other preferences.

    4. Explore Built-in Features:
  • Use Ctrl+P to quickly navigate files.
  • Access the Command Palette with Ctrl+Shift+P or Cmd+Shift+P for advanced commands.

Emmets :

Emmet is a built-in toolkit in Visual Studio Code (VS Code) designed to enhance productivity for developers, especially those working on front-end web development. It enables you to write HTML, CSS, and other code faster by expanding abbreviations into complete code structures.

Emmet Usage
Parent Selector : (>) This creates a parent element with a child element inside of it.(Nested child elements)
Child combinator: (^) It indicates that an element is a direct child of another element. This means the element on the right side of the > will be nested inside the element on the left side, as a direct child.
Sibling Combinator: (+) is used to insert an element next to a previous element as a sibling. It allows you to create a sibling element on the same level in the HTML hierarchy.
Multiplication Operator: (*) It is used to repeat an element or a group of elements a specified number of times. This allows you to create multiple instances of a particular element quickly.
Curlu braces:{} It can be used to group elements or content. This is particularly useful when you need to repeat a set of nested elements or modify content in a specific way.
Attribute: [ ] allows you to quickly add HTML attributes to elements using specific shorthand notation.
