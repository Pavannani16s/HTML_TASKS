FRONT END DEVELOPMENT

Front-end development focuses on creating the visual and interactive parts of a website or application that users directly interact with. Here's an overview to help you understand the key components and skills involved:

Core concepts of Front End Development :

Technology Core Skills Logo
HTML
  • Structure and content of a webpage.
  • Elements like headings, paragraphs, links, images, forms, etc.
    		•
    CSS
  • Styling the webpage (e.g., colors, layouts, fonts).
  • Frameworks: Bootstrap, Tailwind CSS.
    		•
    JAVASCRIPT
  • Adds interactivity (e.g., sliders, modals, form validation).
  • Modern frameworks/libraries: React, Angular, Vue.js.
    		•
    REACTJS
  • For building user interfaces, particularly for single-page applications.
  • developed by Facebook and is now maintained by Meta.
    		•
    TYPESCRIPT
  • It is a strongly-typed, object-oriented programming language that builds on JavaScript by adding optional static typing.
  • developed by Microsoft and is designed to make JavaScript development more robust, scalable, and error-resistant, especially for large-scale projects.
    		•
    ANGULAR
  • It is a strongly-typed, object-oriented programming language that builds on JavaScript by adding optional static typing.
  • developed by Microsoft and is designed to make JavaScript development more robust, scalable, and error-resistant, especially for large-scale projects.
    		•
    BOOTSTRAP
  • A popular open-source front-end development framework for building responsive and mobile-first websites.
  • It provides pre-designed components like navigation bars, modals, and grids.
    		•
    VUE.JS
  • It is a progressive JavaScript framework used for building user interfaces and single-page applications (SPAs).
  • It is popular for its simplicity, flexibility, and ability to integrate seamlessly with other projects or libraries.
    		•
    EJS
  • It is a simple templating language for generating HTML markup with plain JavaScript. It is widely used in server-side applications, especially in Node.js, to create dynamic web pages.
  • EJS allows developers to embed JavaScript logic directly into HTML, making it powerful and flexible for generating dynamic content.
